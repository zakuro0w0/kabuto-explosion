# kabuto-explosion

## やりたいこと
- マリオRPGのミニゲーム「爆裂カブトムシ」的な2Dシューティングゲームを自分で作ってみる
- 使ってみたい技術駆動で実現方法を考える

## 使ってみたい技術
- プログラミング言語: Rust
- ゲームエンジン: [Bevy](https://github.com/bevyengine/bevy)
- wasm化してwebブラウザ上でも動かしてみたい
